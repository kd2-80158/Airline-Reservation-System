Getting Started

1. Clone the repository.
2. Install the required dependencies if they need any updates(all the required libraries and files are pre-installed in node modulus)
3. Open the Frontend Application in VS Code.
4. Run the application using the command npm start.
5. Access the application at http://localhost:3000 in your web browser.
6. Open the spring-boot-backend folder in the STS 3.9 software used for java development.
7. Update the Maven and run the app by clicking spring boot application.
8. If everything goes up fine then boom you have connected frontend with backend application.
